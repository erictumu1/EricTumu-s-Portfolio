Pop Fruity

Pop Fruity is an interactive beverage website that showcases five refreshing soda flavors in a fun and engaging way.

This project was built using the Next.js framework along with technologies such as Three.js, GSAP, Prismic, Drei, and Tailwind CSS to deliver smooth animations, dynamic 3D visuals, and a modern design.

Pop Fruity features immersive 3D soda cans, fluid motion effects, and a playful interface that highlights each flavor.

Coach Career

Coach Career is a full stack SaaS AI-powered web application designed to provide personalized career advice and support to users.

This full stack app was built using the Next.js framework along with technologies such as Tailwind CSS, Shadcn UI, Spline, Neon Postgres, and Clerk for authentication, payment, and subscription management.

Coach Career features a one on one AI career chat powered by multiple LLM agents, a resume analysis tool, job compatibility feedback, and career roadmap generation. It also includes dynamic visualizations using React Flow and utilizes imagekit.io for storing and retrieving user resume data. The frontend is deployed on Vercel, with version control maintained via GitHub.

TuMunch

TuMunch is a full stack food delivery app showcasing various ready to eat food items available for ordering.

Time T

Time T is a full stack AI Trip planner that generates personalized trip advice for any destination worldwide with the provided criteria(Budget, number of days of travel, number of people) etc.

This full stack app is being built using React JS, MongoDB, TailwindCSS, shadcn/ui, Firebase, and the Google Gemini AI.

Time T features various features such as Google place autocomplete, intelligent place filtering according to budget, a log of previous destination searches among many other features available on the App.
